## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

- Go to frontend folder

```sh
cd frontend
```

- npm

```sh
npm i
```

- npm start

```sh
npm start
```

- And then press a

```sh
a
```

### Installation

Clone the repo

```sh
git clone https://github.com/ThNam203/WorkWise
```
